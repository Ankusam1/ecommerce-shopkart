# Ecommerce Shopkart

A Django based E-commerce web application where users can browse products, search items, add them to cart and place orders.

## Features

- User Authentication (Login / Register)
- Product Listing
- Product Detail Page
- Search Products
- Category Filter
- Add to Cart
- Wishlist
- Order Management
- Dark Mode UI
- Admin Panel for Product Management

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Installation

Clone the repository:

git clone https://github.com/Ankusam1/ecommerce-shopkart.git

Navigate to project folder:

cd ecommerce-shopkart

Install dependencies:

pip install -r requirements.txt

Run the server:

python manage.py runserver
